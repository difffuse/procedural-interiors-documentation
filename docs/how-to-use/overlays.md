# Overlays

Visual guides and interactive overlays that help you create perfect interior layouts.

## Overview

Overlays provide real-time visual feedback during interior generation, showing guidelines, measurements, and placement hints to ensure professional results.

## Enabling Overlays

### Access Overlay Controls
1. Open the **Procedural Interiors** panel (press `N` in 3D Viewport)
2. Navigate to the **Overlays** section
3. Toggle individual overlay types on/off

### Overlay Types

#### Construction Guidelines
- **Grid Lines**: Snap-to-grid for precise placement
- **Room Boundaries**: Visual outline of room perimeter
- **Circulation Paths**: Suggested movement corridors
- **Furniture Zones**: Optimal areas for different furniture types

#### Measurement Overlays
- **Dimensions**: Real-world measurements for all objects
- **Clearances**: Minimum spacing requirements
- **Scale References**: Human figure for size comparison
- **Proportional Guides**: Golden ratio and design proportion helpers

#### Asset Placement Hints
- **Snap Points**: Magnetic attachment points for furniture
- **Orientation Guides**: Suggested furniture facing directions
- **Grouping Indicators**: Related objects that work well together
- **Style Consistency**: Visual cues for matching design styles

## Working with Overlays

### Basic Workflow
1. **Enable Guidelines**: Turn on construction guidelines
2. **Set Room Boundaries**: Define your space limits
3. **Add Circulation**: Mark primary movement paths
4. **Place Major Elements**: Use snap points for large furniture
5. **Fine-tune Details**: Adjust with measurement overlays

### Overlay Customization

#### Visual Settings
- **Opacity**: Adjust overlay transparency (0-100%)
- **Color Coding**: Customize colors for different overlay types
- **Line Thickness**: Adjust visibility for different screen sizes
- **Text Size**: Scale measurement text for readability

#### Behavior Settings
- **Auto-Hide**: Overlays disappear when not needed
- **Snap Strength**: How strongly objects snap to guidelines
- **Update Frequency**: Real-time vs. on-demand overlay updates
- **Selective Display**: Show overlays only for selected objects

## Overlay Categories

### Room Layout Overlays

#### Floor Plan Grid
- **Purpose**: Align walls and major elements
- **Grid Size**: Adjustable from 10cm to 1m increments
- **Snap Behavior**: Objects automatically align to grid
- **Visual Style**: Subtle lines that don't interfere with design

#### Circulation Analysis
- **Traffic Flow**: Visualize movement patterns
- **Bottlenecks**: Identify potential congestion areas
- **Accessibility**: ADA-compliant pathway requirements
- **Emergency Egress**: Clear exit routes

### Furniture Placement Overlays

#### Comfort Zones
- **Seating Areas**: Optimal conversation distances
- **Work Surfaces**: Ergonomic desk and counter heights
- **Storage Access**: Clear zones around cabinets and closets
- **Lighting Coverage**: Illumination area indicators

#### Style Coordination
- **Color Harmony**: Visual cues for complementary colors
- **Scale Relationships**: Proportional sizing guides
- **Design Periods**: Historical style consistency helpers
- **Brand Matching**: Coordinate furniture from same manufacturers

### Technical Overlays

#### Structural Elements
- **Load-Bearing**: Identify structural walls and supports
- **Utilities**: Plumbing, electrical, and HVAC considerations
- **Building Codes**: Compliance with local regulations
- **Accessibility**: Universal design requirements

#### Lighting Analysis
- **Natural Light**: Window placement and sun angles
- **Artificial Light**: Fixture coverage and shadow analysis
- **Task Lighting**: Specific illumination requirements
- **Ambient Levels**: Overall room brightness distribution

## Advanced Overlay Features

### Custom Overlay Creation
Create your own overlay types for specific needs:

1. **Define Parameters**: Set measurement criteria
2. **Choose Visual Style**: Colors, lines, and text formatting
3. **Set Behavior**: When and how overlay appears
4. **Save as Preset**: Reuse in future projects

### Overlay Presets

#### Residential Presets
- **Living Room**: Seating arrangements and entertainment zones
- **Kitchen**: Work triangle and appliance clearances
- **Bedroom**: Furniture placement and circulation
- **Bathroom**: Fixture spacing and accessibility

#### Commercial Presets
- **Office**: Workstation layouts and meeting areas
- **Retail**: Customer flow and product display
- **Restaurant**: Table spacing and service paths
- **Healthcare**: Equipment access and patient flow

### Integration with Other Tools

#### Asset Browser
- Overlays show compatible assets for current selection
- Preview how assets will fit before placement
- Highlight recommended combinations

#### Parametric Controls
- Overlays update automatically with parameter changes
- Visual feedback for constraint violations
- Real-time validation of design decisions

## Best Practices

### Overlay Usage Tips
1. **Start Simple**: Begin with basic guidelines, add complexity gradually
2. **Layer Information**: Use multiple overlay types together
3. **Context Switching**: Different overlays for different design phases
4. **Performance**: Disable unused overlays for better viewport performance

### Common Workflows

#### Initial Layout Phase
- Enable room boundaries and circulation paths
- Use grid overlays for major element placement
- Focus on proportional relationships

#### Detailed Design Phase
- Add measurement overlays for precision
- Use style coordination helpers
- Enable comfort zone indicators

#### Final Review Phase
- Technical overlays for code compliance
- Lighting analysis for illumination quality
- Accessibility checks for universal design

## Troubleshooting Overlays

### Performance Issues
- **Reduce Overlay Complexity**: Disable unused overlay types
- **Lower Update Frequency**: Switch to manual refresh mode
- **Simplify Visual Style**: Use thinner lines and smaller text

### Visibility Problems
- **Adjust Opacity**: Make overlays more or less transparent
- **Change Colors**: Use contrasting colors for better visibility
- **Scale Text**: Increase text size for high-resolution displays

### Accuracy Issues
- **Check Units**: Ensure scene units match overlay measurements
- **Calibrate Snap**: Adjust snap strength for your workflow
- **Update References**: Refresh overlay data after major changes
